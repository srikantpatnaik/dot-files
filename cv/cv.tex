\section{Projects}


\textbf{\raggedright\large 8051 programmer for Linux}
{\raggedleft\textit{(Oct 2008 - Feb 2013)}}\\

A first open hardware and software programmer tool for Atmel 8051 based ISP
microcontrollers. Written in C and can support other compatible devices as well.

This software was popular until the USB based programmers
stormed the market.
\href{https://sourceforge.net/projects/linux-isp-89sxx/}{https://sourceforge.net/projects/linux-isp-89sxx}\\

%----

\textbf{\raggedright\large Dual boot on Aakash tablet} {\raggedleft\textit{(Jun 2011 - Mar 2013)}}\\

An affordable tablet by MHRD, Govt. of India to enhance education in rural areas.\\
Customized and ported Linux as secondary OS on 7" ARM tablet. It required device driver
modifications, hardware debugging, and a touch optimized lightweight distribution.

The pilot project has been tested with 1,00,000 devices.
\href{http://aakashlabs.org/gnu}{http://aakashlabs.org/gnu}\\
\href{https://github.com/androportal/linux-on-aakash}{https://github.com/androportal/linux-on-aakash}\\

%---

\textbf{\raggedright\large Programming on Android} {\raggedleft\textit{(May 2012 - Aug 2013)}}\\

A programming environment with C, Cpp, Python and Scilab on Aakash Android tablet. \\
Developed a tiny customized Linux chroot environment to bootstrap packages. A webapp was
used for the frontend.

The pilot project has been tested with 1,00,000 devices.
\href{http://aakashlabs.org/ac/project/1}{http://aakashlabs.org/ac/project/1}\\
\href{https://github.com/androportal/installer}{https://github.com/androportal/installer}\\

%---

\textbf{\raggedright\large FOSSEE Laptop} {\raggedleft\textit{(Feb 2013 - Aug
2015)}}\\

An affordable Linux based laptop for students with popular scientific and
educational packages. \\
The development started with reverse engineering a tv setupbox
Kernel source code. The display and audio drivers were patched to bring the
basic linux running.

The pilot project has been tested with 1,000 devices.

\href{http://laptop.fossee.in}{http://laptop.fossee.in}\\
%\href{https://github.com/FOSSEE/FOSSEE-netbook-kernel-source}{https://github.com/FOSSEE/FOSSEE-netbook-kernel-source}\\
%---

\textbf{\raggedright\large Debugging TRU64 server}
{\raggedleft\textit{(May 2008)}}\\

Worked with Crompton Greaves as a freelancer at Konkan railway to setup auto sync
between Windows application server and old TRU64 Unix server. \\
